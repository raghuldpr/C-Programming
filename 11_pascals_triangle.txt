PROGRAM 11: Pascal's Triangle

Each number is the sum of the two numbers directly above it.
The coefficient is calculated using the formula: coef = coef * (i - j) / (j + 1)

Pattern Output (n = 5):
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1

#include<stdio.h>

int main()
{
    int i, j, n, coef = 1;

    printf("Enter rows: ");
    scanf("%d", &n);

    for(i = 0; i < n; i++)
    {
        coef = 1;

        for(j = 0; j <= i; j++)
        {
            printf("%d ", coef);
            coef = coef * (i - j) / (j + 1);
        }

        printf("\n");
    }

    return 0;
}

/*
Sample Input:
Enter rows: 5

Sample Output:
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
*/
